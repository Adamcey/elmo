Installing ELMO

Ensure you have the following dependencies installed:

  • Ruby version 1.8.7
  • MySQL 5.0 or higher

Then follow these steps:

  1. Get the source code from http://code.google.com/p/elmo/source/checkout
  2. Install the required gems by running bundle install
  3. Create an empty database and accompanying user for use by the app.
  4. Copy config/database.yml.example to config/database.yml and edit this file in order to configure your database settings.
  5. Create the database structure by running rake db:schema:load RAILS_ENV=production
  6. Seed the database by running rake db:seed RAILS_ENV=production
  7. Copy config/initializers/local_config.rb.example to config/initializers/local_config.rb and edit this file in order to configure some important app settings specific to your setup.
  8. Start the server (you can run rails server to use the built-in WEBrick test server).
  9. Open the app in a browser and login using username: super password: changeme
  10. Change the 'super' account password to something more secure.
